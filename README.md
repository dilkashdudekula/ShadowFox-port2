# TravelX - Travel Agency Website

## Objective

To design and develop a modern, responsive travel agency website that promotes popular travel destinations, tour packages, and travel services through an attractive and user-friendly interface using only HTML5 and CSS3.

## Features

* **Hero Section:** Full-screen banner with an eye-catching travel background, welcome message, and call-to-action button.
* **Navigation Menu:** Simple and responsive navigation with links to Home, Destinations, Packages, About, and Contact.
* **Popular Destinations:** Attractive destination cards showcasing famous tourist locations.
* **Why Choose Us:** Highlights the agency's key services, including luxury accommodations, affordable packages, and 24/7 customer support.
* **Tour Packages:** Displays multiple travel packages with pricing and package details.
* **About Us Section:** Provides an overview of the travel agency and its mission.
* **Testimonials:** Customer review section to build trust and credibility.
* **Contact Section:** User-friendly contact form for travel inquiries and bookings.
* **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices using CSS Flexbox, Grid, and Media Queries.
